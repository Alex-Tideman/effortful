== README

This app is called Effortful, which is a community improvement application for those in need. Members can create Efforts where they will do an activity or action that improves where they live. Other users can sponsor members and vote on their progress. If the Effort is a success, a volunteer will pick up a non-monetary reward from the sponsors and deliver it to the member using What3Words, which allows delivery without an address.

This app using Google Omniauth, Google Maps/Directions/Javascript API and What3Words API. It's test suite uses Rspec.
